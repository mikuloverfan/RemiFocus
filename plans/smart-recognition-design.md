# 智能卡片识别系统设计

## 📐 核心思路：大小卡片混合检测

### 小卡片（English）
```
一条列表项 = 一张卡片
- ==mediate==: v. 调停...     → 卡片
- ==radiant==: adj. 发光的...  → 卡片
```

### 大卡片（西综）
```
一个标题 = 一张卡片
### 【看到啥】动脉血压
* **【想到啥】** → 四大条件...  → 属于该卡片的内容
* **【记住啥】** `血充心射`     → 卡片助记
```

### 混合模式（一个笔记里两种都有）
```
## 一词多义    ← 表格（可做卡片也可跳过）
| word | ... |   → 大卡片

### 1-40       ← 小卡片区
- ==word==: ... → 小卡片
```

## 🏗 提取架构

```
┌──────────────────────────────────────┐
│          笔记内容 (markdown)           │
└────────────────┬─────────────────────┘
                 │
                 ▼
┌──────────────────────────────────────┐
│  格式检测器 FormatDetector            │
│                                      │
│  ├─ 检查是否有 ### 【看到啥】 → 大卡片  │
│  ├─ 检查是否有列表项 pattern → 小卡片   │
│  ├─ 检查是否有表格 → 表格卡片           │
│  └─ 返回 detectedFormats[]           │
└────────────────┬─────────────────────┘
                 │
                 ▼
┌──────────────────────────────────────┐
│  多通道提取器 MultiPassExtractor      │
│                                      │
│  Pass 1: 提取大卡片（标题级别）         │
│  ├─ 扫描 ### / ## 标题                │
│  ├─ 判断是否为"大卡片模式"             │
│  │  (有 【想到啥】/多行内容)           │
│  └─ 提取为 ClozeCard                  │
│                                      │
│  Pass 2: 提取小卡片（行级别）           │
│  ├─ 扫描列表项 (- / *)                │
│  ├─ 跳过已属于大卡片的行               │
│  ├─ 匹配格式: ==highlight== / **bold**│
│  │         word:def / word-def 等     │
│  └─ 提取为 SimpleCard                 │
│                                      │
│  Pass 3: 提取表格卡片                  │
│  ├─ 扫描 markdown 表格                │
│  ├─ 表头=字段，每行=一张卡片           │
│  └─ 提取为 TableCard                  │
└────────────────┬─────────────────────┘
                 │
                 ▼
┌──────────────────────────────────────┐
│  卡片合并器 CardMerger                │
│                                      │
│  ├─ 去重：相同 word 合并到同一卡片     │
│  ├─ 同一个 word 可属多个卡组           │
│  └─ 输出 mergedCards[]               │
└────────────────┬─────────────────────┘
                 │
                 ▼
           deck.json 保存
```

## 📦 卡片数据结构

```typescript
// 小卡片
{
  "mediator": {
    "meaning": "n. 调停者",
    "deck": ["红宝书测试/必考词/1-40"],
    "cloze": undefined,
    "mnemonic": undefined
  }
}

// 大卡片（Cloze）
{
  "【看到啥】动脉血压形成条件": {
    "meaning": "完整内容...",
    "deck": ["西综/生理学/血压/动脉血压形成条件"],
    "cloze": [
      { "hint": "靶心 → ", "answer": "四大条件..." },
      { "hint": "结论：", "answer": "缺一不可..." }
    ],
    "mnemonic": "血充心射，外周阻大A弹"
  }
}

// 表格卡片
{
  "radiate": {
    "meaning": "熟义：辐射 | 生义：发散",
    "deck": ["红宝书测试/一词多义"],
    "tableData": { "熟义": "辐射", "生义": "发散" }
  }
}
```

## ⚙️ 设置页识别规则（可开关）

| 规则 | 描述 | 默认 |
|------|------|------|
| `==高亮==` | `==word==: def` Obsidian 高亮格式 | ✅ on |
| `**加粗**` | `**word**: def` 加粗格式 | ✅ on |
| 短横线 | `word - def` | ✅ on |
| 冒号 | `word: def` | ✅ on |
| 括号挖空 | `(word): def` | ✅ on |
| 西综大卡片 | `### 【看到啥】` 整段提取 | ✅ on |
| 表格卡片 | `\| word \| ... \|` 每行一张 | ❌ off |
| 纯高亮 | `==word==` 单独一行 | ❌ off |

### 按文件夹预设

用户可以设置某个文件夹使用特定的卡片模式：

```
📂 English/ → 小卡片模式（默认）
📂 西综/    → 大卡片模式（默认）
📂 通用笔记/ → 混合模式（自动检测）
```

## 🎮 Session 渲染策略

### 小卡片（当前实现不变）
```
正面: word
背面: meaning
```

### 大卡片（Cloze 模式，新增）
```
正面: 【看到啥】动脉血压形成条件
      ─────────────────────────
      【想到啥】靶心 → ______    ← 点击揭示
      【别选啥】混淆→ ______    ← 点击揭示

      💡 助记: 血充心射...      ← 始终显示

揭示后:
      【想到啥】靶心 → 四大条件：循环血量...
      【别选啥】混淆→ 正解：心脏射血是动力...
```

逐段揭示（Progressive Disclosure）：
1. 先显示标题 + 提示
2. 点击"想到啥"区域 → 揭示答案
3. 点击"别选啥"区域 → 揭示正解
4. 助记始终显示
