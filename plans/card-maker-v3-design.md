# Card Maker v3 — 完整重设计

## 1. 新交互流程

```
Step 1: 点击 🖊 制卡按钮
        ↓
Step 2: 工具栏进入"选择模式"（按钮高亮、鼠标变为十字）
        ↓
Step 3: 在笔记中**多次点击/框选** — 每次选择一段文本
        ↓
Step 4: 完成选择后，点击 ✅ 确认（或再次点击 🖊 退出选择模式）
        ↓
Step 5: 弹出一个**批量制卡弹窗**，显示所有选中的文本
        ↓
Step 6: 在弹窗中对每个条目进行加工：
           - 自动拆分 word / pronunciation / meaning
           - 点击 ✂ 可对某个条目挖空
           - 点击 ⛓ 可拆分条目
           - 点击 🎲 随机打乱
        ↓
Step 7: 选择卡组 → 点击 💾 保存
```

## 2. 磁吸工具栏

工具栏固定在**当前标签页右上角**（不是屏幕右上角），跟随标签页滚动。

```
┌──────────────────────────────────────────────┐
│  Obsidian 标签页: 红宝书测试/必考词/1.md      │
│                                               │
│  ### 1-40                    ┌──────────────┐│
│  - ==tow==: ...              │ 🧠 制卡器    ││
│  - ==take==: ...             │              ││
│  - ==talent==: ...           │ [🖊 制卡]    ││ ← 点击进入选择模式
│  - ==target==: ...           │ [✂ 挖空]    ││
│                              │ [🔍 预览]   ││ ← 卡片预览
│  ← 选择模式: 点击文本选择      │ [📦 1-40 ▾] ││
│    已选: 5 段                │ [💾 保存]   ││
│                              └──────────────┘│
└──────────────────────────────────────────────┘
```

实现方式：用 `position: absolute` 相对于编辑器容器定位，而不是 `position: fixed`。

### 自动拆分规则

对于 `==word==: \`[pron]\` def` 格式：

```
原始: ==tow==: `[təʊ]` v.拖，拉，牵引

拆分:
  word: tow
  pron: [təʊ] 
  meaning: v.拖，拉，牵引
```

## 5. 自动识别增强

对于 `- ==word==: \`[pron]\` meaning` 格式，提取器需要：

| 字段 | 来源 |
|------|------|
| `word` | `==...==` 内的内容 |
| `meaning` | 音标后的所有内容 |
| `pronunciation` | 作为 `meaning` 的一部分保留 |

提取正则增强：
```
/^\s*[-*]\s+==(.+?)==:\s*`\[([^\]]+)\]`\s*(.+)$/
  → word = $1
  → pron = $2
  → meaning = $3  (音标词保留在meaning中)
```
