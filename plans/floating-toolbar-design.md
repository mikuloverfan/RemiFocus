# 浮动制卡工具栏 — 架构设计

## 核心理念

制卡器不再是大弹窗，而是一个**浮动在笔记上方的迷你工具栏**，用户在 Obsidian 编辑器中直接选择文本 → 点工具栏按钮 → 创建卡片。

## 布局

```
┌──────────────────────────────────────────────────────┐
│ Obsidian 编辑器标签页                                   │
│                                                       │
│  ### 1-40                    ┌──────────────────────┐ │
│  - ==radiate==: v.发出...    │ 🧠 制卡器 [—] [✕]   │ │ ← 浮动工具栏
│  - ==radiant==: adj.发光...  │                      │ │   始终置顶
│  - ==object==: n.物体...     │ 🖊 制卡  ✂ 挖空      │ │
│                              │ 🔓去挖空 ↩撤回 ⭐优先级│ │
│  ← 直接在笔记中选择文本 →     │ 📦移动 🗑删除         │ │
│                              │                      │ │
│                              │ ──────────────────   │ │
│                              │ 📦 卡组: [1-40 ▾]    │ │
│                              │ [+新建]              │ │
│                              │                      │ │
│                              │ 📋 已创建: 3 张      │ │
│                              │ ★ radiate            │ │
│                              │ ☆ radiant            │ │
│                              │ ☆ object             │ │
│                              └──────────────────────┘ │
└──────────────────────────────────────────────────────┘
```

## 交互流程

```
1. 用户打开笔记（如 红宝书测试/必考词/1.md）
2. 按 Ctrl+P → "制卡器" 或 右侧栏点 [🛠 制卡]
3. 浮动工具栏出现在屏幕右上角

制卡流程：
  1. 在笔记编辑器中选中文本（例如 "radiate: v.发出"）
  2. 点击工具栏 🖊 制卡
  3. ⚡ 自动创建卡片（无弹窗，直接保存到当前卡组）
  4. 工具栏底部列表更新显示新卡片

挖空流程：
  1. 在笔记中选中要隐藏的答案文本
  2. 点击 ✂ 挖空
  3. ⚡ 自动创建 Cloze 卡片
```

## 浮动工具栏功能

| 按钮 | 功能 |
|------|------|
| 🖊 制卡 | 选中文本 → 自动解析 word/meaning → 创建卡片 |
| ✂ 挖空 | 选中文本 → 创建 Cloze 卡片 |
| 🔓 去挖空 | 选中已创建的卡片 → 移除挖空 |
| ↩ 撤回 | 撤销上次操作 |
| ⭐ 优先级 | 标记选中卡片为优先 |
| 📦 移动 | 选中卡片 → 移动到其他卡组 |
| 🗑 删除 | 删除选中卡片 |
| 📦 卡组选择 | 下拉切换当前卡组 |
| [+新建] | 创建新卡组 |

## 清理/忽略规则

| 规则 | 说明 |
|------|------|
| 忽略前导 `- ` | 自动去掉列表项标记 |
| 忽略 `==` | 自动去掉高亮标记 |
| 忽略 `**` | 自动去掉加粗标记 |
| 忽略多余空格 | 压缩连续空格 |
| 截断过长文本 | 正面最多 80 字符 |

## 数据结构简化

不再需要 `cardMaker.ts` 中的复杂三栏布局 + 卡片列表预览。
精简为：
- `ui/cardMaker.ts` — 浮动工具栏组件（~200行）
- 保存逻辑在 `main.ts` 中
