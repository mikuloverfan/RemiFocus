## 自定义规则系统

点击 [⚙ 编辑规则] → 弹出规则编辑器：

```
┌────────────────────────────────────────────┐
│ ⚙ 提取规则                                 │
│                                            │
│ ┌── 内置规则（可开关） ──────────────────┐ │
│ │ ☑ ==(.+?)==: `[([^\]]+)]` (.+)       │ │
│ │   → word=$1  pron=$2  meaning=$3      │ │
│ │ ☑ ==(.+?)==: (.+)                     │ │
│ │   → word=$1  meaning=$2               │ │
│ │ ☑ **(.+?)**: (.+)                     │ │
│ │   → word=$1  meaning=$2               │ │
│ │ ☑ (.+?): (.+)                         │ │
│ │   → word=$1  meaning=$2               │ │
│ └────────────────────────────────────────┘ │
│                                            │
│ ┌── 自定义规则 ──────────────────────────┐ │
│ │  [添加新规则]                           │ │
│ │                                         │ │
│ │  #1: (regex) → word=$1 meaning=$2      │ │
│ │     [测试] [编辑] [删除]               │ │
│ └────────────────────────────────────────┘ │
│                                            │
│  测试文本: [==test==: 测试              ] │
│  匹配结果: word=test  meaning=测试        │
│                                            │
│  [保存规则] [恢复默认] [✕ 关闭]          │
└────────────────────────────────────────────┘
```

## 规则格式

每条规则包含：
```typescript
interface ExtractRule {
  id: string;
  name: string;
  regex: string;        // JavaScript 正则表达式
  enabled: boolean;
  fields: {             // 捕获组到字段的映射
    word?: number;       // 捕获组索引
    meaning?: number;
    pronunciation?: number;
  };
  builtin: boolean;      // 是否为内置规则
}
```

## 存储

规则保存在 `RemiFocusSettings` 中：
```typescript
settings.customRules: ExtractRule[]
```
