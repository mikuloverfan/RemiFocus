# 优雅批量制卡 UI 设计

## 工作流

```
1. 复制文本（从笔记/PDF/网页）
         │
         ▼
2. 粘贴到制卡器输入区
         │
         ▼
3. 自动识别 → 每行一张卡片
   自动拆分 word / pron / meaning
         │
         ▼
4. 检查 → 改错 → 点击 ✂ 自动挖空（识别 ==highlights==）
         │
         ▼
5. 🏷 分配卡组（每个卡片可选不同卡组 / 批量分配）
         │
         ▼
6. ✨ 保存 → 完成！
```

## 核心交互

| 操作 | 方式 |
|------|------|
| 添加卡片 | 粘贴文本 / 手动输入 / 从笔记选择 |
| 自动识别 | 每行 → 一张卡片，自动 split |
| 编辑卡片 | 点击 word/meaning 直接编辑 |
| 挖空 | 点击 ✂ → 自动检测 ==高亮== 并创建 cloze |
| 分配卡组 | 每张卡片独立下拉选择，也可批量选 |
| 删除 | 🗑 删除单张 / 清空全部 |
| 预览 | 🔍 打开预览窗 |
| 保存 | 每张卡片写入各自卡组 |

## 实施

修改 `ui/cardMaker.ts` 中的 `showBatchModal` 方法即可。
