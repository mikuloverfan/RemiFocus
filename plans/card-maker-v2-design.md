## 核心改进

### 1. 渲染预览（不是源代码）

用 Obsidian 的 `MarkdownRenderer.render()` 渲染笔记内容，用户看到的是和 Obsidian 编辑器中一样的渲染效果（加粗、高亮、列表样式等）。

检测到的卡片在渲染视图中用**高亮底色 + 角标**标记：
```
 radiate ── v.发出...       ← 绿色高亮 + 🃏 角标
 radiant ── adj.发光的...    ← 可点击选中
```

### 2. 框选（Rubber Band Selection）

```
鼠标按下 → 拖拽 → 矩形选框出现 → 松开 → 选中框内卡片

┌──────────────┐
│ ✂ radiate    │  ← 选中状态（蓝色边框）
│ ✓ radiant    │
│              │
│ object       │  ← 未选中
└──────────────┘

选中后 → 工具栏亮起批量操作：
[⭐ 批量标星] [📦 移动到...] [🗑 批量删除]
```

### 3. 去挖空工具

`🔓 去挖空` — 选中一张有 cloze 数据的卡片 → 点击 → 移除 cloze 结构，恢复为普通卡片。

### 4. 卡组管理器（右侧栏）

树形结构展示所有卡组，与文件系统一致：

```
红宝书测试/                    ← 文件夹
├──必考词/                     ← 子文件夹
│  ├── 1-40 (40张)           ← 卡组名+卡片数
│  ├── 41-80 (25张)
│  └── 81-120 (0张)
├──基础词/
│  └── Unit1 (30张)
│
血液循环/                      ← 另一个文件夹
└──动脉血压/
   ├── 收缩压 (1张)
   ├── 舒张压 (1张)
   └── 脉压 (1张)
```

操作：
- 点击卡组 → 筛选只显示该卡组的卡片
- 右键 → 重命名 / 删除卡组
- 拖拽卡片到目标卡组
- 新建卡组（在当前位置）

### 5. 批量移动

```
选中多张卡片 → 点击 [📦 批量移动] → 弹出选择框:
┌─────────────────────┐
│ 移动到哪个卡组？      │
│                     │
│ 📂 红宝书测试/       │
│    ├─ 必考词/1-40   │
│    ├─ 必考词/41-80  │ ← 点击选择
│    └─ 基础词/Unit1  │
│                     │
│  [确认移动] [取消]    │
└─────────────────────┘
```
