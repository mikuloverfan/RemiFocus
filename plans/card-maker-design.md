# Card Maker — 可视化制卡器

## 架构

**位置：** Obsidian 右侧边栏（ItemView），替换现在的 QuickView（QuickView 合并到 Dashboard 里）

**触发：** 点击 🧠 Ribbon 图标 → 打开制卡器（显示当前笔记）

## UI 设计

```
┌─────────────────────────────┐
│ 🧠 制卡器                    │ ← 标题
│ 📄 红宝书测试/必考词/1.md    │ ← 当前笔记
├─────────────────────────────┤
│ 📦 卡组: [1-40 ▾] [+新建]   │ ← 卡组选择
├─────────────────────────────┤
│ 🔍 自动检测: 40 个卡片       │ ← 概览
│ ✅ 全部确认 | ❌ 全部忽略    │
├─────────────────────────────┤
│                             │
│ 笔记内容（可交互视图）         │
│                             │
│ ### 1-40                    │
│ ┌─────────────────────────┐ │
│ │ ✅ ==radiate==: ...     │ │ ← 绿色 = 已确认
│ │ ✅ ==radiant==: ...     │ │
│ │ ☐ ==object==: ...       │ │ ← 灰色 = 未选择
│ │ ✅ ==objective==: ...   │ │
│ └─────────────────────────┘ │
│                             │
│ (更多行...)                  │
│                             │
├─────────────────────────────┤
│ [💾 保存选中卡片到 deck.json] │ ← 底部按钮
│ 已选: 38 张                  │
└─────────────────────────────┘
```

## 交互逻辑

1. **打开制卡器** → 自动读取当前笔记 → 运行检测
2. **检测结果高亮显示**：
   - 小卡片（`- ==word==: def`）：绿色背景
   - 大卡片（`### 【看到啥】`）：蓝色背景
   - 表格行（`| word | ... |`）：紫色背景
3. **用户自由操作**：
   - 点击卡片 → 切换选中/取消
   - 拖选文字 → 右键 → "创建卡片"
   - 下拉选择要存入的卡组
   - "新建卡组" 按钮 → 输入名称
4. **保存**：
   - 点击保存 → 写入 deck.json
   - Notice 提示保存结果
