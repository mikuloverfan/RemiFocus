```
┌─────────────────────────────────────────────────────────────┐

├────────────────────────┬────────────────┬──────────────────┤
│                        │                │                  │
│  笔记预览区 (50%)       │  工具栏 (磁吸)   │  卡牌管理区      │
│                        │                │  (30%)           │
│  ### 1-40              │ ┌──────────┐   │                  │

│  - ==radiant==: adj..   │ │ ✂ 挖空    │   │  ┌──────────┐  │

│                        │ │ 🔗 合并    │   │  │ 🃏 radiant │  │
│  == 可点击选择文本 ==    │ │ ⛓ 拆分    │   │  │ 🃏 object  │  │
│                        │ │ 🏷 标签    │   │  │           │  │
```

## 🧲 磁吸工具栏（始终可见）

| 工具 | 图标 | 功能 |
|------|------|------|
| 🖊 制卡 | `[A]` | 选中笔记中的文本 → 创建卡片（选中=正面，再选=反面） |
| ✂ 挖空 | `[...]` | 选中文本 → 创建 Cloze 挖空（答案隐藏） |
| ↩ 撤回 | `↩` | 撤销上一步操作 |
| 🔗 合并 | `⛓` | 选中多张卡片 → 合并为一张 |
| ⛓ 拆分 | `✂` | 选中一张卡片 → 按条件拆分 |
| 🏷 标签 | `#` | 为卡片添加标签/分类 |
| 📂 移动 | `→` | 移动卡片到其他卡组 |
| 🔍 预览 | `👁` | 切换笔记预览/隐藏 |
| 🎯 优先级 | `⭐` | 标记为重点卡片（高优先级） |

## 🎴 卡片操作

### 点击笔记文本 → 制卡流程

```
1. 在笔记预览区选中文本
   → 工具栏亮起 [🖊 制卡]
   
2. 点击 [🖊 制卡]
   → 弹出制卡小窗:
      正面: [选中的文本]
      背面: [空，等待输入或继续选择]
      卡组: [1-40 ▾]
      [确认] [取消]

3. 点击 [确认]
   → 卡片出现在右侧卡牌管理区
   → 笔记中该文本高亮（绿色背景）
```

### 创建 Cloze 挖空

```
1. 选中要隐藏的答案文本
2. 点击 [✂ 挖空]
3. 选中文本变为 {{c1::答案}} 格式
4. 卡片自动生成 cloze 数据
5. 学习时该部分默认隐藏
```

### 拖拽管理

```
卡牌管理区：
┌──────────────────┐
│ 📦 卡组: 1-40    │ ← 下拉切换
│                  │
│ 🃏 radiate    ⭐ │ ← 点击星标=高优先级
│ 🃏 radiant    ⭐ │
│ 🃏 object        │ ← 可拖拽到其他卡组
│ 🃏 objective     │
│                  │
│ [+ 新建卡组]     │ ← 创建新组
└──────────────────┘
```

## 💾 数据存储

```
制卡器保存 → deck.json 结构:

{
  "radiate": {
    "meaning": "v.发出...",
    "deck": ["红宝书测试/必考词/1-40"],
    "state": "new",
    "priority": 1,           ← 手动制卡优先级更高
    "source": "manual",       ← "auto" 或 "manual"
    ...
  }
}
```

- `priority` 字段：手动制卡 priority=1（高优先级），自动扫描 priority=0
- `source` 字段：标记卡片来源（manual/auto）
- 排序时 priority=1 的卡片排在前面
