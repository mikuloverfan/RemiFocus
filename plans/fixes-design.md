# 修复方案

## 问题 1：插件关闭后工具栏残留

**原因：** `cardMaker` 字段是 `public`，`onunload` 中没有调用 `destroy()`。

**修复：** `main.ts` 的 `onunload()` 中添加 `this.cardMaker?.destroy()`

## 问题 2：磁吸工具栏只吸附右边缘

**修复：** 拖拽松手时检测距离 4 个边缘的距离，吸附到最近的一个：

```
松手时检测:
  距右边缘 < 80px → 吸附右边缘
  距左边缘 < 80px → 吸附左边缘
  距上边缘 < 40px → 吸附顶部
  距下边缘 < 40px → 吸附底部
```

在 Session 界面顶部添加卡片级统计：
